# 기록앞에

오늘 경기에서 달성 가능한 프로야구 기록을 살펴보는 광고 없는 웹 MVP입니다.

화면 설계와 기능 검수는 모바일 뷰를 우선 기준으로 진행합니다.

## 현재 기능

- 날짜별 경기 일정 탐색
- 경기별, 타자/투수별 기록 필터
- 선수명과 기록명 검색
- 선수, 구단, 구장별 달성 가능 기록 전환
- 목표까지 남은 수치와 달성 가능성 자동 계산
- 선수 시즌 현황과 최근 5경기 상세 보기
- 선호팀 설정과 브라우저 저장
- 데스크톱/모바일 반응형 화면
- 기본 다크 테마와 모바일 화면 전환형 하단 탭

## 데이터 범위

현재 일정과 선수 기록은 모두 UI와 계산 로직을 검증하기 위한 샘플입니다. KBO 또는 구단의 공식 데이터, 로고, 기사, 예상 기록 게시물을 수집하거나 복제하지 않습니다.

정식 서비스 전환 시에는 허가받은 API 또는 데이터 공급 계약을 통해 `GAMES`와 `RECORD_INPUTS` 입력값을 교체해야 합니다. 달성 여부와 남은 기록 계산은 앱 내부의 `getRecordState` 함수가 담당합니다.

## 실행

```bash
npm install
npm run dev
```

배포용 빌드는 `npm run build`로 생성합니다.

`main` 브랜치에 변경 사항이 올라오면 GitHub Actions가 정적 빌드를 생성해 GitHub Pages에 자동 배포합니다.
